# טבלת הכיבוד 🥦 · בחסות חסלט

משחק קצר, כיפי ו(בכוונה) בלתי אפשרי לניצחון.

זוהי **טבלת ההרשמה לכיבוד** של מסיבת סוף השנה בגן. אתם אמורים לרשום את
שמכם ליד מה שתביאו — חבילת פיתות, שרוול כוסות חד״פ... אבל איך שאתם מתחילים,
כל ההורים האחרים מתנפלים על הטבלה ותוך שניות תופסים את **כל** הדברים הקלים.
נשארת לכם רק שורה אחת ריקה:

> 🥦 **כרובית ללא חרקים**

אתם רושמים שם את שמכם — והטוויסט: זו דווקא כרובית של **חסלט**, ירק נקי בלי
חרקים ובלי שאריות חומרי הדברה. אז אולי דווקא יצאתם מנצחים. 😌

**הטעם הנקי של הטבע 🌱**

## 🎮 איך משחקים
1. לוחצים "תרשמו אותי" (או על שורה כלשהי בטבלה).
2. צופים חסרי אונים איך כל השורות מתמלאות בשמות של הורים אחרים.
3. רושמים את שמכם בשורה היחידה שנשארה — הכרובית.
4. מקבלים כרטיס תוצאה לשיתוף בוואטסאפ.

## 🛠️ טכנולוגיה
HTML / CSS / JavaScript טהור — בלי תלויות, בלי build. מתארח חינם ב‑GitHub Pages.

```
index.html      · מבנה הדף והטבלה
styles.css      · עיצוב, אנימציות, RTL
game.js         · לוגיקת ההרשמה (הטבלה שנחטפת מתחת לאף)
assets/logo.jpg · הלוגו של חסלט
```

## ✏️ החלפת הלוגו
רוצים להחליף את הלוגו? פשוט החליפו את הקובץ `assets/logo.jpg` בקובץ שלכם
(עדיף ביחס דומה, רקע לבן). אין צורך לשנות קוד.
