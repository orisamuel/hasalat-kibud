# המאמן לכיבוד 🏆 · בחסות חסלט

משחק קצר, כיפי ו(בכוונה) בלתי אפשרי לניצחון.

נרשמת לכיבוד של מסיבת סוף השנה בגן? יש לך **מאמן אישי** שמבטיח שתחטוף את
הדברים הקלים — חבילת פיתות, שרוול כוסות חד״פ... אבל ברגע שאתה נכנס לטבלה,
כל ההורים האחרים כבר הקדימו אותך. הפריטים נחטפים, חלקם אפילו **בורחים מהעכבר**,
ובסוף נשאר לך רק דבר אחד:

> 🥦 **כרובית ללא חרקים**

והטוויסט — זו דווקא כרובית של **חסלט**: ירק נקי, בלי חרקים, בלי שאריות חומרי
הדברה. אז אולי דווקא יצאת מנצח/ת. 😌

**הטעם הנקי של הטבע 🌱**

## 🎮 איך משחקים
1. לוחצים "אני מוכן/ה".
2. המאמן סופר לאחור — ואתם מנסים לחטוף פריט קל.
3. ...ונכשלים. כל סיבוב מהירים יותר. עד שנשארת רק הכרובית.
4. מקבלים כרטיס תוצאה לשיתוף בוואטסאפ.

## 🛠️ טכנולוגיה
HTML / CSS / JavaScript טהור — בלי תלויות, בלי build. מתארח חינם ב‑GitHub Pages.

```
index.html      · מבנה המסכים
styles.css      · עיצוב, אנימציות, RTL
game.js         · לוגיקת המשחק (המאמן שאי אפשר לנצח)
assets/logo.jpg · הלוגו של חסלט
```

## ✏️ החלפת הלוגו
רוצים להחליף את הלוגו? פשוט החליפו את הקובץ `assets/logo.jpg` בקובץ שלכם
(עדיף ביחס דומה, רקע לבן). אין צורך לשנות קוד.
