# שיבוץ לכיבוד 🥦 · בחסות חסלט

משחק קצר, חצוף ו(כמעט תמיד) בלתי אפשרי לניצחון.

מתחיל במוקאפ של **קבוצת ווצאפ** (רנדומלי בכל פעם): הגננת/המדריכה שולחת קישור
לטבלת שיבוץ הכיבוד, מישהי כותבת "רגע אי אפשר למלא", מישהו "תני הרשאת עריכה",
"תנסו עכשיו"... ואז לוחצים על הקישור ומגיעים לטבלה — ועד שהבנת איך פותחים,
**כל ההורים כבר חטפו את הכל**.

בסוף תמיד נשארות **שתי אופציות בלבד**:
> 🍄 משהו הזוי לגמרי (משתנה בכל פעם) · 🥗 ומגש ירקות חתוכים

ואם כבר נתקעת עם ירקות — שיהיו של **חסלט**: נקיים מחרקים, מפוקחים משאריות
חומרי הדברה. (כן, היינו חייבים פה משפט שיווקי. מישהו צריך לממן את המשחק 🤷)

**פעם ב~8 משחקים** המערכת מתרשלת ומשאירה לך משהו קל אמיתי — אם תהיה מהיר.

## 🎮 הזרימה
1. צופים בשיחת הווצאפ (אפשר לדלג) → לוחצים על הקישור.
2. הטבלה נפתחת והכל נחטף תוך שניות.
3. נשארות 2 האופציות המופרכות → כרטיס סיום חצוף + שיתוף בוואטסאפ.

## 🛠️ טכנולוגיה
HTML / CSS / JavaScript טהור — בלי תלויות, בלי build. מתארח חינם ב‑GitHub Pages.

```
index.html      · מסך הווצאפ, הטבלה וכרטיס הסיום
styles.css      · עיצוב ה‑UI של ווצאפ, הטבלה והתוצאה (RTL)
game.js         · מנוע השיחה הרנדומלי + לוגיקת השיבוץ שנחטף
assets/logo.jpg · הלוגו של חסלט
```

נבדק אוטומטית בדפדפן (Chrome headless): מנוע השיחה, מסלול ההפסד ומסלול הזכייה.

## ✏️ החלפת הלוגו
מחליפים את `assets/logo.jpg` בקובץ שלכם (רקע לבן, יחס דומה). אין צורך לגעת בקוד.
